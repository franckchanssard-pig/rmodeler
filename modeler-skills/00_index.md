# Pigment Modeler Skills (KB‑based)

Purpose: concise, actionable skills for building and maintaining high‑performance Pigment models.

Use these files as a knowledge base for a modeler agent. Each file focuses on a core competency.

## Contents
- `10_formula_syntax_modifiers.md` — formula grammar, modifiers, and composition patterns
- `20_performance_optimization.md` — scoped calculations, profiling, and performance insights
- `30_lists_dimensions_blocks.md` — list types, dimensions, and structural modeling rules
- `40_security_access_rights.md` — security folder, access rights rules, ACCESSRIGHTS usage
- `50_libraries_multi_app.md` — library usage and sharing across applications
- `60_scenarios_formula_groups.md` — formula groups per scenario and versioning
- `70_api_workflows.md` — practical API steps for lists, metrics, formulas, boards

## How to use
- Start with **formula syntax** and **lists/dimensions** for fundamentals.
- Apply **performance** guidance early; it saves time later.
- Use **security** and **libraries** when moving to production or multi‑app setups.
